INTRODUCTION
------------

The Custom Timezone module gives the option to see the Site location and the 
current time for the timezone configured.

REQUIREMENTS
------------

No special requirements.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.
   See https://www.drupal.org/node/1897420 for further information.


CONFIGURATION
-------------

 * Configure the Custom Timezone settings at (/admin/config/timezone).


Standard usage scenario
--------------------------------------------------------------------------------
1. Install the module.
2. Open admin/config/timezone.
3. Make the changes to the Country, City and Time zone as needed.
4. Configure the 'Site location and current time' block as needed.
    4.1. It will give the current date and time of the selected timezone.
